

```
var a = {n:1};  
var b = a; // 持有a，以回查  
a.x = a = {n:2};  
alert(a.x);// --> undefined  
alert(b.x);// --> {n:2}
```

`a = a.x = {n:2}`和`a.x = a = {n:2}`的结果是一致的

原因:
- 找到 a 和 a.x 的指针。如果已有指针，那么不改变它。如果没有指针，即那个变量还没被申明，那么就创建它，指向 null。
a 是有指针的，指向 {n:1}；a.x 是没有指针的，所以创建它，指向 null。
- 然后把上面找到的指针，都指向最右侧赋的那个值，即 {n:2}。
